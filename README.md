# T9 Decrypter
A T9 Decryper in nasm assembly x86_64 (intel 64 bits)<br>
Made from scratch.

This is an entry in [the second devcode challenge](docs/challenge.pdf)<br>
[View notes](docs/notes.md)

## Usage

### Linux
Install nasm with

    $ sudo apt-get update
    $ sudo apt-get install nasm

and run project with

    $ make all
    or
    $ make build
    $ dist/main
