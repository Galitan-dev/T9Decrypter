# ThePriceIsRight
A "The Price Is Right" game in nasm assembly x86_64 (intel 64 bits)
Made from scratch.
## Usage

### Linux
Install nasm with
```
$ sudo apt-get update
$ sudo apt-get install nasm
```
and run project with
```
$ make all
or
$ make build
$ dist/main
```
